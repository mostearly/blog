使用WSL安装NVM

    记录/1 天前/6 分钟前/1

    安装NVM

    使用Root或者常用的用户

    # 移动至安装目录
cd /usr/local
# 新建文件夹
mkdir nvm
# 移动至src文件夹
cd src
sudo git clone https://github.com/creationix/nvm.git

    编辑~/.bashrc文件

    vim ~/.bashrc

    在里面追加

    export NVM_DIR=/usr/local/nvm
source /usr/src/nvm/nvm.sh

    然后重新登录后验证下

    nvm --version

    正常输出就没问题

    安装NodeJS

    查看可安装的NodeJS版本

    nvm -ls-remote

    安装稳定版本12.16.3

    nvm install 12.16.3

    安装常用的Yarn包

    安装yarn

    npm -g i yarn

    然后进入项目重新安装一次项目
