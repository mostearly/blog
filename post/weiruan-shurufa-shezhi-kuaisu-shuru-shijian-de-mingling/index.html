微软输入法设置快速输入时间的命令

    教程/1 年前/5 个月前/1

    作为一款Windows10亲儿子的输入法怎么能没用快速输入时间的命令？

    首先在设置中找到“时间和语言”然后找到在菜单栏“语言”一项中点击首选语言，这时候会出现一个“选项”的按钮，点进去就是输入法的设置。。。(前面步骤过多)，直接在桌面菜单栏时间旁边找到"中"或“英”的图标然后右击设置就进去了

    在“词库和自学习”中找到“添加或编辑自定义短语”，进去后点击“添加”然后输入下面的数据即可，key对应拼音、addr对应位置、code对应短语

    切记编辑时只能编辑code然后复制进去，否者会变成静态的文字而不是动态的时间

    year

    {
    key: "year",
    addr: 2,
    code: "%yyyy%"
    preview: "2019"
}

    now

    {
    key: "now",
    addr: 2,
    code: "%yyyy%-%MM%-%dd%"
    preview: "2019-02-12"
}

    now

    {
    key: "now",
    addr: 3,
    code: "%yyyy%-%MM%-%dd% %HH%:%mm%:%ss%"
    preview: "2019-02-12 22:54:07"
}

    now

    {
    key: "now",
    addr: 4,
    code: "%MM%月%dd%日"
    preview: "02月12日"
}

    now

    {
    key: "now",
    addr: 5,
    code: "%HH%:%mm%"
    preview: "22:55"
}

    month

    {
    key: "month",
    addr: 2,
    code: "%yyyy%-%MM%"
    preview: "2019-02"
}

    rq

    {
    key: "rq",
    addr: 2,
    code: "%yyyy%年%MM%月%dd%日"
    preview: "2019年02月12日"
}

    sj

    {
    key: "sj",
    addr: 2,
    code: "%HH%:%mm%:%ss%"
    preview: "22:57:40"
}
